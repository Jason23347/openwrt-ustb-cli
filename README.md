# openwrt-ustb-cli

基于 OpenWrt 官方 SDK 的 USTB 校园网命令行客户端 `ustb-cli` 打包工程，用于在 OpenWrt 上构建并分发 `ustb-cli` 的 `.ipk` 软件包。  
本仓库同时提供 GitHub Actions 工作流，可自动为指定 OpenWrt 版本与架构（当前为 `rockchip-armv8`，版本 `24.10`）构建并上传 `.ipk` 到 Release。

> 上游项目：[`ustb-cli-c`](https://github.com/Jason23347/ustb-cli-c) —— USTB 校园网命令行客户端
